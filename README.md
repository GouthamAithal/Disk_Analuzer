# Disk Analyzer Dashboard

A real-time disk health monitoring dashboard that analyzes CrystalDiskInfo logs and visualizes SMART metrics.

## Features
- Parses CrystalDiskInfo logs
- Health scoring engine
- Risk prediction system
- Historical tracking
- Trend visualization graphs
- Interactive Streamlit dashboard

## Tech Stack
Python, Streamlit, Matplotlib, Regex Parsing

## How to Run

pip install -r requirements.txt
streamlit run app.py
